1. **不要当混蛋**  
在你对其他GSF成员采取任何行动之前,请自问:“如果别人对我这样做,我会有什么感受?” 如果答案是“不好”,那就不要做.

本帖每个章节都会说明该政策的“预期结果”.这告诉你政策希望实现的目标.即使你没有违反明文规定,也可能违背了政策意图.这里不是法庭,我们不是法官,你也不是能靠钻漏洞取胜的律师.不要试图用规则文字游戏挑战制度,那不会有好结果.

{{% expand title="市场与经济政策" %}}
**期望结果:**  
Goonswarm Federation 的市场应当以有利的价格和充足的数量,为本联盟成员提供相对于敌对势力的经济优势.敌对方不应从我们的经济实力中获利.

**具体指导:**
1. 不得向敌对势力销售商品,尤其是在我们的领地或邻近区域.禁止在 NPC空间站 销售任何商品.
2. 禁止在 GSF 市场上“重挂单”(relisting).  
“重挂单”包括但不限于:为了让自己的物品卖得更快而大量购买便宜的挂单.  
举例:假设你以 400k 出售 2000 个 Vigil,而有人以 300k 出售 200 个.若你清空那 200 个便宜 Vigil,让市场上只剩你的挂单,这仍然是“重挂单”.
即使你声称那 200 个 Vigil 现在在你仓库里,也不行.你剥夺了其他成员购买便宜 Vigil 的机会.不要这么做.
3. 不得将联盟市场中的物品取走后转卖至联盟控制范围外的市场.  
但你可以自由处置自己生产的物品,或他人通过收购单卖给你的物品.  
该规则的目的在于防止有人将 Delve 市场上他人辛苦生产、低价出售的物品大量运往 Jita 牟利.  
若你想补充盟友市场库存,请从 Jita 采购,而非从我们的市场搬货.
4. 价格利润率不受限制.  
如果你觉得某商品价格太高,请自行进口并压价竞争.

**处罚措施:**  
若违反上述政策,你的所有角色与小号公司(alt corp)将被加入 Goonfucker ACL(访问控制名单),至少 90 天.
名单解除每季度统一处理一次,不单独提前释放.若你的违规时间恰巧跨两个季度,那是你自己的问题.

**加入 Goonfucker ACL 将导致以下惩罚:**

- 公司办公室费用:100,000,000 ISK(正常为 10,000,000)
- 克隆费用:10,000,000 ISK(正常为 0)
- 工业活动税率:10%(正常为 0.25%)
- 市场活动税率:11.5%(正常为 2%)
- 再处理税率:50%(正常为 5%)
- 跳桥液态臭氧费用:500 ISK/单位(正常为 150 ISK)


{{% notice style="tip" title="注意" %}}
如果你不希望你的公司或联盟成员被列入 Goonfucker ACL,务必确保所有成员阅读并遵守本政策.
该政策适用于 Goonswarm Federation 及帝国盟友(Imperium),因为我们的市场是共同体.

{{% expand title="攻击友军" %}}
**期望结果:**  
成员应当避免对联盟或盟友成员造成任何形式的损害.任何涉及“蓝方”（即联盟或盟友）的敌对行为,均被视为严重违规.

**具体规定:**
1. 禁止攻击蓝方成员的舰船、无人机或可部署物.  
包括但不限于:在异常点、任务区、矿带、ESS、堡垒等区域主动锁定并开火.

2. 禁止以“误击”为借口的蓄意攻击.  
若确实发生误击,请立即在本地频道（local）或舰队频道（fleet）中公开道歉,并赔偿损失.

3. 禁止通过间接方式造成蓝方损失.  
例如:在 ESS 抢夺阶段故意释放智能炸弹（smartbomb）,或在友军舰队中使用范围性武器（AOE weapon）导致己方受损.

**处罚措施:**  
- 轻微事件（如误击并及时赔偿）将由公司内部处理.  
- 严重事件（如故意击杀蓝加、拒绝赔偿、屡犯者）将被移交 GIT 调查,可能导致:
  - 联盟开除
  - 永久列入 Goonfucker ACL
  - 公司罚款或解散

**期望结果:**

2. 若你使用范围性武器造成蓝方舰船损失,你必须全额赔偿受害者的损失.  
若拒不赔偿,将被视为“友军互射”行为,按上条处理.  
3. 在大型 PvE 活动（例如防御 ESS、抵御 NPC 入侵等）中,若因使用范围性武器导致舰队成员损失,责任由使用者承担.

**处罚措施:**  
- 第一次违规:书面警告,并要求赔偿.
- 第二次违规:加入 Goonfucker ACL,至少 30 天.
- 屡犯者:永久禁止参与联盟 PvE 活动.

1. 先到先得原则,若某人已进入异常点并开始击杀 NPC,则该异常点归其所有.后来者不得以任何方式干扰.

3. 禁止使用探针（combat probe）扫描并干扰他人PvE,任何此类行为均视为蓄意破坏.  
4. 若多个玩家同时进入同一异常点,以“先锁定并造成伤害者”为准.若双方争议,可截图并提交 GIT 仲裁.  
5. 异常点内的可部署物（如 MTU）归放置者所有.未经许可不得摧毁或掠夺他人 MTU.  

**处罚措施:**  
- 第一次违规:书面警告,并要求赔偿.
- 第二次违规:加入 Goonfucker ACL,至少 30 天.
